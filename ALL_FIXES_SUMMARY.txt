            🔥 CREATE FIRESTORE (2 MINUTES)
═══════════════════════════════════════════════════════════════

This will make artist names and saves work permanently!

1. Open Firebase Console
   👉 https://console.firebase.google.com/project/nextwave-music-sim

2. Create Firestore Database
   • Click "Build" → "Firestore Database"
   • Click "Create database"
   • Select "Start in test mode"
   • Choose region (closest to you)
   • Click "Enable"

3. Enable Authentication
   • Click "Build" → "Authentication"
   • Click "Get Started"
   • Enable "Email/Password"
   • Enable "Anonymous"

4. Test Again!
   • Restart app: flutter run -d windows
   • Create NEW account (old one wasn't saved)
   • Complete onboarding
   • Dashboard now shows YOUR name! ✅
